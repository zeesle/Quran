I setup GitHub for hostinger, now how bring the repo on hostinger

Kodee
Use the Websites button I shared, then go to Add website → Import Git Repository.
Connect your GitHub account, pick the repo, review the build/start commands, and click Deploy.
If the repo is private, generate an SSH key in Websites → Manage → GIT, copy it, and add it to GitHub under Settings → SSH and GPG keys.
After that, Hostinger will pull the code and deploy it to a temporary subdomain.
